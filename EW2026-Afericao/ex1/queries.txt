# Ex1: Quantos automóveis foram reparados?

db.repairs.countDocuments()

# Ex2: Quantos automóveis da marca "Cadillac" foram reparados?

db.repairs.find({ "viatura.marca": "Cadillac" }).count()

# Ex3: Que marcas de automóveis já apareceram na oficina (ordenadas alfabeticamente e sem repetições)?

db.repairs.distinct("viatura.marca").sort()

# Ex4: Qual a distribuição por tipo de reparação? (por exemplo: quantas mudanças de óleo, quantas mudanças de filtro, ...)

db.repairs.aggregate([{ $unwind: "$intervencoes" }, { $group: { _id: "$intervencoes.nome", total: { $sum: 1 } } }, { $sort: { total: -1 } } ])

# Ex5: Qual a distribuição de clientes por ano?

db.repairs.aggregate([{ $group: { _id: { $substr: ["$data", 0, 4] }, totalClientes: { $sum: 1 } } }, { $sort: { _id: 1 } }])
